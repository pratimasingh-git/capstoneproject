**Capstone Assignment – Multilevel Inheritance in Python**

Python program using multilevel inheritance

Person → Student → Exam

to calculate and display the total marks of a student.

 (a) Base Class: Person

-->This class stores basic personal details like name and age.

--> It also contains a method to display these details.

 (b) Derived Class: Student

--> This class inherits the Person class.

--> It adds extra information such as the course the student is enrolled in.

 (c) Derived Class: Exam

--> This is the final class in the chain.

--> It inherits from Student.

--> It stores marks of three subjects and calculates the total marks.

 (d) Object Creation and Display

--> An object of the Exam class is created with sample data.

--> The program prints all details:

name, age, course, marks, and total.
